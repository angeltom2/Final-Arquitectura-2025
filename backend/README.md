# Backend - Konrad Gourmet (skeleton)

Tecnologías:
- Node.js + Express
- Sequelize (MySQL)
- Patrón Mediator: un servicio simple registra handlers y dispatch con mediator.send()
- Dockerfile incluido

Para iniciar localmente:
1. Copiar .env.example a .env y ajustar variables.
2. npm install
3. npm run dev

Endpoints de ejemplo:
- GET /api/health
- GET /api/example
